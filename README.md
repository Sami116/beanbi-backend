## Bean智能 BI 项目介绍
基于React + Spring Boot+ MQ+AIGC的智能数据分析平台。
区别于传统Bl，用户只需要导入原始数据集、并输入分析诉求，就能自动生成可视化图表及分析结论，实现数据分析的降本增效。
AIGC: AI生成内容
